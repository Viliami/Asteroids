# Asteroids
3D C++/OpenGL Asteroids clone.
Currently a work in progress.

###Dependencies
- SOIL
- libassimp
- freeglut3
- C++11 or higher

###Screenshot
![Alt text](screenshot.png "Work in progress")

###How to compile and run
Simply type
```
make
```
and then run the program by typing
```
./test
```
